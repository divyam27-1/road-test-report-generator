{\footnotesize\ttfamily void remove\+Duplicates (\begin{DoxyParamCaption}\item[{std\+::vector$<$ std\+::string $>$ \&}]{vec,  }\item[{const std\+::vector$<$ QString $>$ \&}]{ref = {\ttfamily \mbox{\hyperlink{mainwindow_8cpp_a8a5477800793b213aaa44ba632bbaf27}{all\+\_\+exp\+\_\+qstr}}} }\end{DoxyParamCaption})}



Removes duplicates from a vector and sorts it based on a reference vector. 

This function removes duplicate elements from the input vector {\ttfamily vec}. It then sorts {\ttfamily vec} based on the order of elements in the reference vector {\ttfamily ref}. If an element in {\ttfamily vec} is not found in {\ttfamily ref}, it keeps its relative order. In our case, this function is used to manage the variable "{}tracked\+\_\+files"{}. The {\ttfamily tracked\+\_\+files} variable is the system\textquotesingle{}s internal tracking system for which experiment types are saved, and the order of experiments with which the report must be generated, as specified by the {\ttfamily all\+\_\+experiments} variable.


\begin{DoxyParams}{Parameters}
{\em vec} & The vector from which duplicates are to be removed and which is to be sorted. This parameter is passed by reference and will be modified by the function. In our case this is the {\ttfamily tracked\+\_\+files} variable. \\
\hline
{\em ref} & The reference vector based on which {\ttfamily vec} is to be sorted. It has a default value of {\ttfamily all\+\_\+exp\+\_\+qstr}. In our case we reference the {\ttfamily all\+\_\+experiments} variable. \\
\hline
\end{DoxyParams}

Detailed Description

These functions mark a specific directory where the output report must be saved, then invoke the 'Save' operation.

These functions prompt the user for where they wish the output of the report to be generated, then mark it in the swd directory. Then they invoke the respective 'Save' function for whichever experiment is called for the 'Save As'.

Each function opens a directory selection dialog, sets the selected directory as the working directory, displays a message box informing the user about the export location, and then triggers the 'Save' button click event for the corresponding tab.
